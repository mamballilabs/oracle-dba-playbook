What is instance vs Database?
Instance is memory structure + process , database is actual physical files stored on disk.

What is SGA?
system global area - shared memory which is  part of instance , main components are:

| component             | use/functionality                          |
| --------------------- | ------------------------------------------ |
| database buffer cache | stores data blocks from disk               |
| shared pool           | sotres sql + execution plans               |
| redo log buffer       | stores changes before writing to redo logs |
List and explain 5 background processes?
Background processes are:

| Process name           | functionality                        |
| ---------------------- | ------------------------------------ |
| DBWR - database writer | writes data from memory to datafiles |
| LGWR - log writer      | writes log buffer to redo log files  |
| SMON - system monitor  | crash recovery                       |
| PMON - process monitor | cleans failed sessions               |
| CKPT - checkpoint      | updates file headers                 |

what are datafiles, control files, redo logs?  
The physical storage

| files name     | functionality                     |
| -------------- | --------------------------------- |
| Datfiles       | actual data stored here           |
| control files  | metadata about DB structure       |
| Redo log files | record all changes - for recovery |
explain the flow?
1. user runs SQL
2. SQL goes to SGA
3. Data loaded into buffer cache
4. Changes recorded in redo buffer
5. LGWR writes redo logs(first)
6. DBWR writes data to disk(later)
