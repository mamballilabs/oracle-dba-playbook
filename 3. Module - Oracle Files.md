what are datafiles?
datafiles are physical files on the disk which stores actual user data in the form of tables and indexes .

three different files exists on database

| File name      | functionality             |
| -------------- | ------------------------- |
| datafiles      | stores data               |
| control files  | stores db metadata        |
| redo log files | stores change information |

what is control file?
This stores the metadata of database , that is information about database.
database name, datafile and redo log file location and checkpoint information.

what are redo logs?
Every change made to database to logged here. This helps in crash recovery and data protection.

PFILE vs SPFILE?
These are the parameter files types, which tells oracle how to start the instance.

| Types  | format | Editable | used in modern DB |
| ------ | ------ | -------- | ----------------- |
| PFILE  | text   | Manual   | Rare              |
| SPFILE | binary | SQL      | Default           |
 
Full startup dependency flow?
1. Reads Parameter file (SPFILE)
2. It will get SGA allocation details and control file location
3. Starts instance = allocates SGA and starts background processes => NOMOUNT
4. Reads control file => MOUNT stage
5. Opens Data files + Redo log files => OPEN stage
